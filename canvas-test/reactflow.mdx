## Вариант 5: React Flow

Библиотека для создания node-based интерфейсов. Используется для workflow builders, mind maps, диаграмм.

### Плюсы
- Профессиональный вид
- Интерактивные ноды
- Зум и pan встроены
- Можно кастомизировать полностью

### Минусы
- Нужна кастомная интеграция
- Сложнее в настройке
- Требует React компонент

---

### Как бы выглядел код интеграции

```jsx
import ReactFlow from 'reactflow';

const nodes = [
  { id: '1', data: { label: 'Критерии' }, position: { x: 0, y: 0 } },
  { id: '2', data: { label: 'Доказательства' }, position: { x: 200, y: 0 } },
  { id: '3', data: { label: 'Петиция' }, position: { x: 400, y: 0 } },
  { id: '4', data: { label: 'Approval' }, position: { x: 600, y: 0 } },
];

const edges = [
  { id: 'e1-2', source: '1', target: '2' },
  { id: 'e2-3', source: '2', target: '3' },
  { id: 'e3-4', source: '3', target: '4' },
];

export default function Flow() {
  return <ReactFlow nodes={nodes} edges={edges} fitView />;
}
```

<Warning>
Для полноценной интеграции React Flow нужно создать custom React компонент в Mintlify.
</Warning>
