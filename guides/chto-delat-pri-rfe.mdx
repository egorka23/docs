```
1. Cover Letter
   - Номер кейса
   - Дата RFE
   - Краткое содержание ответа

2. Ответ по каждому пункту RFE
   - Цитата из RFE
   - Ваш ответ
   - Список приложенных доказательств

3. Дополнительные доказательства
   - Новые рекомендательные письма
```
