## По критериям EB-1A

<AccordionGroup>
  <Accordion title="Вклад (Original Contributions)" icon="lightbulb">
    Кейсы с этим критерием. [Подробнее](/success-stories/by-criteria/contributions)

    - Геодезист: нацпроект + методические документы
    - Презентер: вклад в развитие области
    - IT предприниматель: оригинальные разработки
  </Accordion>

  <Accordion title="Судейство (Judging)" icon="gavel">
    Кейсы с этим критерием. [Подробнее](/success-stories/by-criteria/judging)

    - Геодезист: судебная экспертиза (очень сильное доказательство)
    - Финтех: судейство на хакатоне
    - IT предприниматель: жюри
    - Software Engineer: 3 судейства на хакатонах
  </Accordion>

  <Accordion title="СМИ (Press)" icon="newspaper">
    Кейсы с этим критерием. [Подробнее](/success-stories/by-criteria/press)

    - Геодезист: публикации + упоминание в книгах
    - Художник-дизайнер: 4 статьи
    - Презентер: публикации в СМИ
    - Software Engineer: 6 публикаций + YouTube
  </Accordion>

  <Accordion title="Высокая зарплата (High Salary)" icon="dollar-sign">
    Кейсы с этим критерием. [Подробнее](/success-stories/by-criteria/salary)

    - Геодезист: справка Росстата + открытые источники
    - Художник-дизайнер: повышенная ЗП
    - Финтех: 2-НДФЛ + дивиденды
    - Software Engineer: высокая ЗП в двух компаниях
  </Accordion>

  <Accordion title="Членство (Membership)" icon="id-card">
    Кейсы с этим критерием. [Подробнее](/success-stories/by-criteria/membership)

    - Геодезист: 3 членства, 2 труднодоступных
    - Художник-дизайнер: вступление во время кейса
    - Художник: клубы
    - Software Engineer: 3 ассоциации
  </Accordion>

  <Accordion title="Критическая роль" icon="star">
    Кейсы с этим критерием

    - Геодезист: подрядчик нефтяной компании
    - Презентер: роли в организациях
    - IT предприниматель: основатель/лидер
    - Маркетинг: критическая роль в проектах
  </Accordion>
</AccordionGroup>
